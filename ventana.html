Moises javascript

En este script utilizo el objeto window del navegador para crear una ventana y luego utilizo unos metodos del objeto window para manipular la ventana creada

Presiona los botones para realziar las diferenes acciones, con respecto a lso botones de movimiento los puede presionar muchas veces seguidas para ir moviendo la ventana

nota: si no visualizar nada al presionar los botones es porque tu navegador bloquea las ventanas emergentes, en ese caso tienes que habiliar las ventanas emergentes para visiualizar este sitio, y luego recargar la pagina y listo, ya podrias utilizar esos botones
